How to use in Desmos:
  1. Upload an image & click Generate Desmos
  2. Click ⎘ Copy All to Desmos
  3. Open desmos.com/calculator
  4. Click an expression box → press Ctrl+V (or Cmd+V on Mac)
  5. Desmos will split each line into a separate expression automatically

Tips for better results: Use images with clear contrast. Lower edge threshold = more detail. More shade bands = richer color.
