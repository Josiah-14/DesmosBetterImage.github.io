New approach: Solid scanline fills
Instead of dots, this generates filled horizontal line segments per color — like a proper raster fill. Much more solid and realistic.

How to paste into Desmos:
  1. Click ⚡ Generate then ⎘ Copy All
  2. Open desmos.com/calculator
  3. Click the first empty expression box
  4. Press Ctrl+V (or Cmd+V)

💡 Tip: Lower resolution = fewer expressions (loads faster). Higher color count = more detail.
