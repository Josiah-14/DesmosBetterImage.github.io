Table + Inequality method (inspired by your QR bookmarklet!)
Instead of one polygon per pixel, this uses one table + one inequality per color.
Each color gets: a table of (x_N, y_N) pixel positions, plus one expression
x_N ≤ x ≤ x_N+1 { y_N-1 ≤ y ≤ y_N } that fills ALL squares at once.

Result: 2 × numColors expressions total instead of thousands of polygons. Loads instantly in Desmos.
